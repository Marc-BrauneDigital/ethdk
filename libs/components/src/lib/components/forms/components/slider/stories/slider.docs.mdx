# Slider [WIP - not ready for use]

The slider component is used to select a value from a range of values. It is a visual representation of a range input.

## Usage

```html
<et-slider-field [formControl]="fg">
  <et-label>Slider</et-label>
  <et-slider />
</et-slider-field>
```

## Rannge slider

```html
<et-slider-field [formControl]="fg">
  <et-label>Range Slider</et-label>
  <et-rangle-slider />
</et-slider-field>
```
