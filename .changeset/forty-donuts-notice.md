Add signal utils for injecting router data. Data can be transformed just like Angular inputs using the `transform` option.
- `injectQueryParams()`
- `injectQueryParam()`
- `injectPathParams()`
- `injectPathParam()`
- `injectRouteData()` 
- `injectRouteDataItem()`
- `injectFragment()`
- `injectRouteTitle()`
